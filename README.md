
# Ranesh Chandra Katta - Portfolio Website

This is my personal portfolio website built with React and TypeScript. 
## Features

- **Responsive Design**: Works seamlessly across all devices
- **Modern UI Components**: Built with shadcn/ui component library
- **TypeScript**: Type-safe development experience
- **Vite**: Fast build tool and development server

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deployment

This website is deployed using GitHub Pages at: https://ranesh.me

### Automatic Deployment
The site automatically deploys when you push to the main branch using GitHub Actions.

### Manual Deployment
To deploy manually:
1. Run `npm install` to install dependencies
2. Run `npm run build` to build the project
3. Run `npm run deploy` to deploy to GitHub Pages

### GitHub Pages Setup
1. Go to your repository settings
2. Navigate to Pages section
3. Set source to "GitHub Actions"
4. The workflow will automatically deploy on push to main branch
